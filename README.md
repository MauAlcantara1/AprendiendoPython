## Español

# Aprendiendo Python

Repositorio de aprendizaje progresivo en Python. Contiene códigos explicados y documentados que cubren desde la sintaxis básica hasta conceptos avanzados como programación orientada a objetos, recursividad e interacción con el sistema operativo.

---

## Descripción

Este repositorio reúne ejercicios y ejemplos escritos y explicados de manera personal, organizados por tema y nivel de complejidad. Cada archivo está pensado para ser leído y comprendido de forma independiente, funcionando como material de referencia y estudio.

El objetivo es construir una base sólida en Python documentando el proceso de aprendizaje desde los fundamentos hasta temas intermedios y avanzados.

---

## Contenido

### Fundamentos

* **Sintaxis básica** → estructura de un programa en Python, indentación y comentarios
* **Variables y tipos de datos** → enteros, flotantes, cadenas, booleanos
* **Expresiones y operadores** → aritméticos, lógicos, de comparación y de asignación

---

### Estructuras de Control

* **`while`** → bucles con condición, casos de uso y precauciones
* **`for`** → iteración sobre secuencias, rangos y colecciones

---

### Funciones

* **Definición y llamada** → `def`, parámetros, retorno de valores
* **Recursividad** → funciones que se llaman a sí mismas, casos base y recursivos

---

### Estructuras de Datos

* **Cadenas** → manipulación, métodos y operaciones comunes
* **Listas** → creación, indexación, métodos y recorridos
* **Diccionarios** → pares clave-valor, acceso, modificación e iteración

---

### Programación Orientada a Objetos (POO)

* **Clases y objetos** → definición, atributos e instanciación
* **Métodos** → funciones dentro de clases, `self`
* **Herencia y encapsulamiento** → reutilización y organización del código

---

### Python y Sistema Operativo

* **Módulo `os`** → navegación de directorios, rutas y operaciones con archivos
* **Módulo `shutil`** → copia y movimiento de archivos
* **Automatización básica** → tareas del sistema operativo desde Python

## ENGLISH

# Learning Python

A progressive learning repository in Python. Contains explained and documented code covering everything from basic syntax to advanced concepts such as object-oriented programming, recursion, and operating system interaction.

---

## Overview

This repository compiles exercises and examples written and explained personally, organized by topic and complexity level. Each file is designed to be read and understood independently, serving as reference and study material.

The goal is to build a solid foundation in Python by documenting the learning process from fundamentals to intermediate and advanced topics.

---

## Contents

### Fundamentals

* **Basic syntax** → structure of a Python program, indentation, and comments
* **Variables and data types** → integers, floats, strings, booleans
* **Expressions and operators** → arithmetic, logical, comparison, and assignment

---

### Control Structures

* **`while`** → condition-based loops, use cases and cautions
* **`for`** → iteration over sequences, ranges, and collections

---

### Functions

* **Definition and calls** → `def`, parameters, return values
* **Recursion** → functions that call themselves, base and recursive cases

---

### Data Structures

* **Strings** → manipulation, methods, and common operations
* **Lists** → creation, indexing, methods, and traversal
* **Dictionaries** → key-value pairs, access, modification, and iteration

---

### Object-Oriented Programming (OOP)

* **Classes and objects** → definition, attributes, and instantiation
* **Methods** → functions inside classes, `self`
* **Inheritance and encapsulation** → code reuse and organization

---

### Python and the Operating System

* **`os` module** → directory navigation, paths, and file operations
* **`shutil` module** → file copying and movement
* **Basic automation** → OS-level tasks from Python
